# Crypto Dice Game 🎲
Crypto Dice Game 🎲

- Knowledge: keccak256, ecsign, ecrecover
- Soldity: ^0.6.6
- Tool:  ethereum-waffle
- UnitTest: Typescript + mocha + chai

# Local Development

The following assumes the use of `node@>=10`.

## Install Dependencies

`yarn`

## Compile Contracts

`yarn compile`

## Run Tests

`yarn test`
